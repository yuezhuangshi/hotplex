# HotPlex 2026 路线图

> **使命**：成为本地 AI 生态系统的终极执行引擎。
> 
> **策略**：AI 优先开发 —— 利用 AI 编程工具 (Claude Code / OpenCode) 以 3-5 倍于传统速度构建 HotPlex。

---

## 🎯 2026 年度目标

| 目标             | 关键结果 (Key Results)                   |
| ---------------- | ---------------------------------------- |
| **开发者采用率** | GitHub Stars 翻倍，SDK 下载量稳步增长    |
| **性能与稳定性** | 生产级 SLA，基准测试报告，全场景边界覆盖 |
| **生态扩展**     | 多语言 SDK 支持，插件系统，社区贡献      |

### 目标用户
- 独立开发者 / 创业团队
- 平台 / 基础设施团队
- CI/CD 自动化场景
